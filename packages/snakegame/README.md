# snakegame

Snake game in JS

## Features

1. [Gym](https://github.com/openai/gym) like game environment, easy to write AI algorithm
2. All class members are protected, easy to extend
3. 0 dependency

## Developing

```bash
npx lerna run dev --scope=@arkfun/snakegame
```

## Publish

```bash
npx lerna publish
```
